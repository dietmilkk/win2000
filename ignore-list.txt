integrar chat de bate-papo com alternativa ja pre pronta

Crie uma extensão Manifest V3 para o Chrome que remova os cabeçalhos 'X-Frame-Options' e 'Content-Security-Policy' do Discord e do YouTube usando declarativeNetRequest, Adicione um script que injete um iframe flutuante, arrastável e com o visual das janela DENTRO do site para carregar esses sites de forma integrada, a ideia é criar um tipo de browser interno dentro do site que consiga fazer de tudo, foque no simples porém funcional, integrar com o ublock origen para bloqueio de ads e melhorar a experiência do usuário

sistema de armazenamento de dados permanente

gerenciador de arquivos publico

Crie uma extensão Manifest V3 para o Chrome e a estrutura de um site (Web OS) integrado à API do Google Drive usando o Google Sign-In para autenticação simplificada através de um menu de configurações. A extensão deve usar 'declarativeNetRequest' e a permissão de 'cookies' para remover restrições de iframe (X-Frame-Options/CSP) do Discord e YouTube. O sistema deve criptografar e sincronizar na conta do Google Drive do usuário: 1) O estado das janelas e arquivos do Web OS e 2) Os cookies de sessão dos iframes, injetando-os automaticamente em computadores diferentes para manter o usuário sempre logado
